# ACS712 Current Sensor Module

## Overview
This project is an implementation of a current sensing module using the ACS712 Hall-effect integrated circuit. 

## Objectives
- Follow a professional design workflow for a current sensor
- Create schematic and PCB layout based on reference designs
- Document component selection and safety considerations for power electronics
